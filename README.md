# Devlink (Mongo/Mongoose project)


![MIT License Badge](https://img.shields.io/badge/License-MIT-blue)


## Description

DevLink is a backend API for a professional developer networking platform. Built with Node.js, Express, and MongoDB, it allows developers to create profiles, publish short career-focused posts, connect with other developers, and react to posts. The API features full CRUD functionality, Mongoose schema validation, subdocuments, virtual fields, and reference-based relationships between collections.
